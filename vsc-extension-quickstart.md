# VS Code Extension Quickstart

Welcome to the **IntelliJ IDEA 2025.3 Theme** for Visual Studio Code!

## What's Included

- `package.json`: Extension manifest and metadata.
- `themes/intellij-idea-2025.3-color-theme.json`: The color theme definition.

## Getting Started

1. Press `F5` to open a new Extension Development Host window.
2. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`).
3. Run `Preferences: Color Theme` and select **IntelliJ IDEA 2025.3 Theme**.

## Making Changes

- Edit the theme file in `themes/intellij-idea-2025.3-color-theme.json`.
- Changes are applied live in the Extension Development Host.

## Testing Your Theme

- Open files in various languages to see syntax highlighting.
- Use `Developer: Inspect Editor Tokens and Scopes` from the Command Palette to examine tokenization.

## Publishing

1. [Package and publish your extension](https://code.visualstudio.com/api/working-with-extensions/publishing-extension).
2. Follow VS Code's publishing guide for details.

## Support

For issues or feature requests, open an issue on the repository.
